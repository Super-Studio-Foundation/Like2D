Tutorial 1: Hello World

Objective: Create your first Like2D application that displays text on screen.

Step 1: Create Project Structure

HelloWorld/
└── main.luau

Step 2: Write the Code

Create main.luau:

-- main.luau - Hello World Tutorial

function onInit()
    -- Set window properties
    setWindowTitle("Hello World Tutorial")
    setWindowSize(800, 600)
    
    print("Hello World tutorial started!")
end

function onDraw()
    -- Clear screen with a nice blue color
    clearScreen(100, 149, 237)  -- Cornflower blue
    
    -- Draw "Hello World" text in the center
    local text = "Hello, Like2D World!"
    local centerX = 400 - (string.len(text) * 12)  -- Rough centering
    local centerY = 300
    
    drawTextDirect(text, centerX, centerY, 48, 255, 255, 255, 255)
    
    -- Draw smaller text below
    drawTextDirect("Press ESC to exit", 320, 350, 24, 255, 255, 100, 255)
end

function onCleanup()
    print("Hello World tutorial ended")
end

Step 3: Run the Application

.\Like2D.exe "HelloWorld"

Expected Result: A blue window with "Hello, Like2D World!" text in white.

Tutorial 2: Basic Movement

Objective: Add keyboard-controlled movement to a simple colored rectangle.

Step 1: Create Project

Movement/
└── main.luau

Step 2: Write the Code

-- main.luau - Basic Movement Tutorial

-- Player object
local player = {
    x = 400,
    y = 300,
    width = 50,
    height = 50,
    speed = 5,
    color = {r = 255, g = 100, b = 100}
}

function onInit()
    setWindowTitle("Movement Tutorial")
    setWindowSize(800, 600)
    
    print("Movement tutorial started!")
end

function onDraw()
    -- Clear screen
    clearScreen(40, 40, 50)
    
    -- Handle input
    if isKeyDown("Left") then
        player.x = player.x - player.speed
    end
    if isKeyDown("Right") then
        player.x = player.x + player.speed
    end
    if isKeyDown("Up") then
        player.y = player.y - player.speed
    end
    if isKeyDown("Down") then
        player.y = player.y + player.speed
    end
    
    -- Keep player in bounds
    player.x = math.max(0, math.min(800 - player.width, player.x))
    player.y = math.max(0, math.min(600 - player.height, player.y))
    
    -- Draw player (as colored rectangle using drawRect)
    drawRect(player.x, player.y, player.width, player.height, 
             player.color.r, player.color.g, player.color.b, 255, true, 0)
    
    -- Draw UI text
    drawTextDirect("Use Arrow Keys to Move", 10, 10, 20, 255, 255, 255, 255)
    drawTextDirect("Position: " .. math.floor(player.x) .. ", " .. math.floor(player.y), 10, 35, 18, 255, 255, 255, 255)
    drawTextDirect("Press ESC to exit", 10, 60, 18, 200, 200, 200, 255)
end

function onCleanup()
    print("Movement tutorial ended")
end

Step 3: Test the Movement

Run the application and use arrow keys to move the red square around the screen.

Tutorial 3: Image Rendering

Objective: Load and display images in your application.

Step 1: Create Project with Assets

ImageDemo/
├── main.luau
└── assets/
    ├── player.png
    └── background.jpg

Step 2: Prepare Assets

  • Create or download a simple player sprite (64x64 pixels recommended)
  • Create or download a background image (800x600 pixels recommended)
  • Save them in the assets/ folder

Step 3: Write the Code

-- main.luau - Image Rendering Tutorial

local player = {
    x = 400,
    y = 300,
    width = 64,
    height = 64,
    speed = 3
}

function onInit()
    setWindowTitle("Image Rendering Tutorial")
    setWindowSize(800, 600)
    
    -- Load assets with error checking
    local playerLoaded = loadImage("assets/player.png")
    local bgLoaded = loadImage("assets/background.jpg")
    
    if not playerLoaded then
        print("Warning: Failed to load player image")
    end
    
    if not bgLoaded then
        print("Warning: Failed to load background image")
    end
    
    print("Image tutorial started!")
end

function onDraw()
    -- Draw background first
    if not renderImage("assets/background.jpg", 0, 0, 800, 600) then
        -- Fallback: draw colored background
        clearScreen(50, 100, 150)
    end
    
    -- Handle movement
    if isKeyDown("Left") then
        player.x = player.x - player.speed
    end
    if isKeyDown("Right") then
        player.x = player.x + player.speed
    end
    if isKeyDown("Up") then
        player.y = player.y - player.speed
    end
    if isKeyDown("Down") then
        player.y = player.y + player.speed
    end
    
    -- Keep player in bounds
    player.x = math.max(0, math.min(800 - player.width, player.x))
    player.y = math.max(0, math.min(600 - player.height, player.y))
    
    -- Draw player
    if not renderImage("assets/player.png", player.x, player.y, player.width, player.height) then
        -- Fallback: draw colored rectangle
        drawRect(player.x, player.y, player.width, player.height, 255, 100, 100, 255, true, 0)
    end
    
    -- Draw UI
    drawTextDirect("Arrow keys to move, ESC to exit", 10, 10, 18, 255, 255, 255, 255)
    drawTextDirect("Player: " .. math.floor(player.x) .. ", " .. math.floor(player.y), 10, 30, 16, 255, 255, 255, 255)
end

function onCleanup()
    print("Image tutorial ended")
end

Tutorial 4: Multi-file Projects

Objective: Learn how to organize your code across multiple files using dofile().

Step 1: Create Two Files

MyProject/
├── main.luau
└── anim.luau

Step 2: Write the Animation Logic

Create anim.luau. We'll define a function and a variable here:

-- anim.luau
anim_speed = 0.5

function get_blink_alpha()
    return math.sin(getTime() / 1000 * anim_speed) * 0.5 + 0.5
end

Step 3: Include it in Main

Now, use dofile() at the very top of main.luau to "import" that logic:

-- main.luau
dofile("anim.luau")

function onInit()
    setWindowTitle("Multi-file Tutorial")
    setWindowSize(800, 600)
end

function onDraw()
    clearScreen(30, 30, 30)
    
    local alpha = get_blink_alpha() -- Called from anim.luau!
    drawTextDirect("BLINKING TEXT", 300, 300, 32, 255, 255, 255, alpha * 255)
end

Why do this? This keeps your main.luau clean and lets you keep specific systems (like player movement, animations, or enemies) in their own files.
