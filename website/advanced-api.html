Advanced API Reference NEW

Complete documentation for Like2D advanced features including physics, video playback, advanced rendering, camera system, mouse input, and JSON support.

Extended Window Settings

setEscapeQuits(boolean)

Sets whether ESC key automatically quits the game. Default: true.

setEscapeQuits(false)

setLetterboxColor(r, g, b)

Changes letterbox bar color when setUseLetterbox(true) is active.

setLetterboxColor(32, 32, 48)

Mouse Input

isMousePressed(button)

Checks if a mouse button is held down. 0=left, 1=right, 2=middle.

if isMousePressed(0) then
    print("Left click")
end

getMousePos()

Returns current mouse position (x, y).

local x, y = getMousePos()

getMouseScroll()

Returns scroll wheel delta (x, y).

local sx, sy = getMouseScroll()

Time Functions

getDeltaTime()

Returns time in seconds since last frame.

local dt = getDeltaTime()

Camera System

setCameraPosition(x, y)

Sets world camera position.

setCameraPosition(player.x, player.y)

getCameraPosition()

Returns current camera position (x, y).

setCameraZoom(zoom)

Sets camera zoom level (1.0 = normal).

getCameraZoom()

Returns current camera zoom level.

worldToScreen(wx, wy)

Converts world coordinates to screen coordinates.

local sx, sy = worldToScreen(500, 300)

Box2D Physics

Body Management

createBody(x, y, type, fixedRotation)

Creates a physics body. type: "dynamic", "static", or "kinematic". Returns body ID.

local ground = createBody(400, 550, "static", true)
local player = createBody(400, 300, "dynamic", false)

destroyBody(bodyId)

Destroys a physics body.

setBodyPosition(bodyId, x, y)

Teleports body to position.

getBodyPosition(bodyId)

Returns body position (x, y).

getPhysicsTransform(bodyId)

Returns table with x, y, and angle (degrees).

local t = getPhysicsTransform(playerBody)
drawRect(t.x - 25, t.y - 25, 50, 50, 255, 100, 100, 255, true, t.angle)

Velocity & Forces

setBodyVelocity(bodyId, vx, vy)

Sets linear velocity (pixels/second).

getBodyVelocity(bodyId)

Returns current velocity (vx, vy).

applyForce(bodyId, fx, fy)

Applies continuous force to body center.

applyImpulse(bodyId, ix, iy)

Applies instant impulse to body center.

applyImpulse(playerBody, 0, -400)

setGravity(x, y)

Sets world gravity (pixels/second²).

setGravity(0, 980)

Shapes

addBoxShape(bodyId, halfWidth, halfHeight, density, friction, restitution, isSensor)

Adds box collision shape to body.

addBoxShape(playerBody, 25, 25, 1.0, 0.3, 0.5, false)

addCircleShape(bodyId, radius, density, friction, restitution, isSensor)

Adds circle collision shape to body.

addCapsuleShape(bodyId, halfHeight, radius, density, friction)

Adds capsule (pill) shape to body.

Advanced Image Rendering

renderImageEx(key, x, y, w, h, angle, flipX, flipY, alpha, originX, originY)

Renders image with rotation, flipping, transparency, and scaling.

  • angle: degrees
  • flipX, flipY: boolean
  • alpha: 0-1 (default 1.0)
  • originX, originY: 0-1 (0.5 = center)
renderImageEx("player", 400, 300, 64, 64, 45, true, false, 0.5, 0.5, 0.5)

renderImageRegion(key, dx, dy, dw, dh, sx, sy, sw, sh, angle, flipX, flipY, alpha, originX, originY)

Renders a sub-rectangle from a sprite sheet.

renderImageRegion("spritesheet", 400, 300, 64, 64, 0, 0, 32, 32, 0, false, false, 1.0, 0.5, 0.5)

getImageSize(key)

Returns loaded image dimensions (width, height).

local w, h = getImageSize("player")

Video Playback

loadVideo(key, filename)

Loads video file.

updateVideo(key, dt)

Updates video frame. Call in onUpdate().

renderVideo(key, x, y, w, h)

Renders current video frame.

playVideo(key, loop)

Starts playback.

pauseVideo(key)

Pauses playback.

stopVideo(key)

Stops and resets video.

seekVideo(key, seconds)

Jumps to time position.

isVideoPlaying(key)

Returns true if playing.

isVideoEnded(key)

Returns true if video finished.

getVideoDuration(key)

Returns total duration in seconds.

getVideoTime(key)

Returns current position in seconds.

unloadVideo(key)

Unloads video from memory.

JSON Module

json.load(filename)

Loads JSON file, returns table and error.

local data, err = json.load("settings.json")

json.save(filename, table)

Saves table to JSON file.

local ok, err = json.save("save.json", {score = 100})

json.encode(table)

Converts table to JSON string.

json.decode(string)

Parses JSON string into table.

Additional Primitives

drawLine(x1, y1, x2, y2, r, g, b, a)

Draws a line.

drawCircle(x, y, radius, r, g, b, a, filled)

Draws a circle.

setDefaultFont(path)

Sets default font for drawTextDirect().

Miscellaneous

quit()

Cleanly closes the game.

drawRect(x, y, w, h, r, g, b, a, filled, angle)

Draws a rectangle. Set filled=true for solid, false for outline. angle in degrees.

drawRect(100, 100, 64, 64, 255, 0, 0, 255, true, 45)

Audio System

Supported formats: WAV, MP3, OGG.

loadSound(key, filename)

Loads a sound file. Use a unique key to reference it later.

loadSound("jump", "assets/jump.wav")
            loadSound("bgm", "assets/music.mp3")

playSound(key, loop)

Plays a loaded sound. Set loop=true for continuous playback.

stopSound(key)

pauseSound(key)

resumeSound(key)

isSoundPlaying(key)

setSoundVolume(key, volume)

Volume: 0.0 (silent) to 1.0 (full).

setMasterVolume(volume)

Global volume control for all sounds.

setSoundPitch(key, pitch)

1.0 = normal, 2.0 = double speed/pitch, 0.5 = half speed/pitch.

unloadSound(key)
